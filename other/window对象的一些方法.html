    # Window对象和方法

    1、打开和关闭一个浏览器的新的窗口，可以指定窗口的尺寸，内容以及是否有按钮条和地址框等属性 javascript window.open(url,target,features,replace) // url表示打开新窗口的页面地址；target表示窗口的名称(可省略)；features表示窗口的特征(可省略)；replace表示是否替换当前窗口

    name: _blank - URL加载到一个新的窗口。这是默认
         _parent - URL加载到父框架
         _self - URL替换当前页面
         _top - URL替换任何可加载的框架集
         name - 窗口名称         
features: 设置窗口的属性
replace： 规定了装载到窗口的 URL 是在窗口的浏览历史中创建一个新条目，还是替换浏览历史中的当前条目
         true - URL 替换浏览历史中的当前条目。
         false - URL 在浏览历史中创建新的条目。
